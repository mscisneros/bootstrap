Line Height

Line spacing is the space between two lines of text, or more specifically, it's the exact distance between two adjacent baselines, and a very important aspect of user experience design because it has a huge impact on the readability of text in a digital device.

Note: Using the same line-height for all text is a very subtle, but common mistake because 1.5 may work great for body copy, but as text gets larger, your line-height should get tighter.

5 rules for line spacing:

  • Aim for about 140%-180% for optimal readability and accessibility. Any smaller and the text will be cramped and difficult to read. Go much larger and the eye can get lost.
  • Limit line length to 70-80 characters. Longer lines of text can confuse the eye, particularly when reading text on a screen. If you must use longer lines, increase the line spacing to help protect readability.
  • Font size should be minimum 16pt. This is a long-standing rule of thumb that still applies.
  • Small fonts need more spacing. Line spacing should actually increase with smaller font sizes. This is because smaller fonts are already more difficult to read, and need more space around them for the eye to easily follow.
  • Check your line spacing when you change font or font size. Different fonts have different maximum heights (x-heights) even when they're the same size in points. So you need to be sure that when you change font, you check to see how your line spacing feels in readability terms.

How to calculate the ideal line spacing

You may need to do some basic arithmetic to find the optimum line spacing. For example, if your font is 18pt, and we're aiming for line spacing of 150%, simply multiply the font size by 1.5 and you have the correct line spacing value: 27pt.
