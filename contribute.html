Contribute

This design system is the result of contributions – large and small – of code, design, ideas, and guidance that also includes feedback, documentation, new designs, and tools.

All you need is a public GitHub account to get started, and typically these projects use a fork and pull request workflow for contributions. In addition, it's common to include a “CONTRIBUTING.md” file, which explains how the maintainers of the project want to handle contributions, and also often contains instructions for setting up the project.
